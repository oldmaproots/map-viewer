# 地図ビューア（04MapViewer）

全国Q地図のような地図タイル切り替えサイトを自作するプロジェクト。
地理院地図・Q地図風の左サイドパネルでレイヤーを選ぶ構成。
家系図プロジェクト（01FamilyTree）とは無関係の独立プロジェクト。

**公開URL: https://oldmaproots.github.io/map-viewer/** （GitHub Pages。リポジトリ: oldmaproots/map-viewer）

## 動かし方

- `python serve.py` で http://localhost:5504 に配信される（キャッシュ無効化つき）。
- 01FamilyTree の `.claude/launch.json` に `map-viewer-server` として登録済み。

## ファイル構成（ビルド不要）

| ファイル | 役割 |
|---|---|
| `index.html` | 画面の骨組み。Leaflet / MapLibre GL をCDNから読み込む |
| `style.css` | 左パネルなどの見た目 |
| `script.js` | 本体。サイドパネル生成・レイヤー管理・背景地図 |
| `konjaku-data.js` | 今昔マップ全313レイヤーの一覧（**自動生成。手で編集しない**） |
| `dem-layers.js` | 数値標高タイルを読んでブラウザ内で地形を描く部品 |
| `timeseries.js` | 年代別写真の時系列スライダー |
| `kumamoto.js` | 都市計画図（熊本県）のGeoJSONレイヤー定義 |
| `moj-style-2026-*.json` | 法務局地図のMapLibreスタイル（Q地図の定義を複製） |
| `data/kumamoto/` | 02KumamotoCityPlanningからコピーした都市計画GeoJSON |

## 収録レイヤーと出典（2026-07-11時点）

1. **新旧の地形図** — 今昔マップ【谷謙二氏】58地域×各年代。
   タイル: `ktgis.net/kjmapw/kjtilemap/{地域}/{時期}/{z}/{x}/{-y}.png`（`{-y}`=Y反転のTMS方式）。
   出典表示必須: 「今昔マップ on the web」((C)谷謙二)
2. **公図・地番図・地名** — 2026年 法務局地図（登記所備付地図）【KotobaMedia】。
   ベクトルタイル `tiles.kmproj.com/mojxml/2026.json` をMapLibre GL(+leaflet-maplibre-gl)で描画。
   ズーム16以上で筆界と地番文字。塗りつぶしなし版もあり。
3. **地形** — 基盤地図情報(標高) DEM1A【Q地図タイル】/ DEM5A【地理院】。
   数値標高タイル（標高=(R×65536+G×256+B)×0.01m、2^23=無効値）をcanvasで解読し、
   赤色立体風・陰影起伏・傾斜量を**ブラウザ内で計算**（`dem-layers.js`）。
   タイルがないズームでは上位ズームから切り出して拡大（オーバーズーム）。
   DEM1A: `qchizu3.xsrv.jp/mapdata/d52001/{z}/{x}/{y}.webp`（512px, 〜z16）
   出典表示: Q地図タイル(測量法に基づく国土地理院長承認(使用)R7JHs10)
4. **年代別の写真** — 時系列表示（ZL14以上）。1928年頃〜最新の28年代
   （ort_1928 / ort_riku10 / ort_USA10 / ort_old10 / gazo1〜4 / airphoto / nendophoto2007〜2024 / seamlessphoto）。
   画面中央のタイルを各年代で試し読みし、写真がある年代を緑の点で表示。
5. **標高・土地の凹凸** — 色別標高図（地理院reliefタイル）と、
   自分で作る色別標高図（最低/最高標高・段数・陰影・データ元を指定してブラウザ内で描く）。
6. **都市計画図 > 熊本県** — 国土数値情報A55のGeoJSON13種（02プロジェクトと同じ配色）。
   表示中に地図をクリックすると当たった区域名をポップアップ表示。
   **右下に凡例**が出る（チェック中のレイヤーのみ・折り畳み可・項目ごとに地図上の表示/非表示を切替）。
   凡例の仕組みは`kumamoto.js`の`initKumamotoLegend()`。非表示は該当項目を透明で描く方式
   （`def._hiddenItems`）で、クリック判定`kumamotoMatchesAt()`も非表示項目を無視する。
7. **背景地図** — 標準/淡色/白地図/空中写真（地理院）。

そのほかの機能:
- **住所・地名検索**（右上）— 国土地理院の住所検索API（無料・キー不要）。02プロジェクトv2と同方式。
- **現在地ボタン**（右下）— ブラウザの位置情報APIで現在地＋誤差円を表示。

## 共通の仕組み

- レイヤーをONにすると左パネル上部の「選択中のレイヤー」に載り、
  透過スライダーと✕（削除）が使える。`registerActiveLayer()`（script.js）。
- 各カテゴリは`<details>`要素のアコーディオン。`buildCategory()`/`buildLayerRow()`が共通部品。
- 表示位置はlocalStorage（キー: `map-viewer-last-view`）に保存。初期表示は熊本周辺。

## 公開にあたっての利用規約の確認結果（2026-07-12確認）

- **今昔マップ**（ktgis.net/kjmapw/note.html）: 出典表示のみで埋め込み利用可・申請不要。
  ただし**1日1000ページビューを超えるようなサイトでの埋め込みは禁止**。個人サイトの想定内だが、
  もしアクセスが増えたら今昔マップのレイヤーを外すこと。タイルの複製・再配信も禁止。
- **Q地図タイル**: 「出典：全国Q地図（国土地理院測量成果）」の出典明示で利用可（qchizu.jp/reprint）。
- **地理院タイル・国土数値情報・KotobaMedia法務局地図**: いずれも出典明示で利用可（attributionに設定済み）。
